Description
    Execute CPU ERoT management related actions.
Required Arguments
    --action    <action>
          Sets action to:
          1 = GetInfo
          2 = UpdateGolden
          3 = Recover
Optional Arguments
    None
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c CpuERotManage --action GetInfo
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c CpuERotManage --action Recover
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c CpuERotManage --action
        UpdateGolden
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c CpuERotManage --action GetInfo --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c CpuERotManage --action
        GetInfo
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c CpuERotManage --action GetInfo
Notice
    1. Currently only support on G1SMH/G1SMH-G platform.
    2. UpdateGolden action will overwrite golden image with current CPU ERoT
    index 0 firmware.
