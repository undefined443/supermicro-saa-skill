Description
    Gets the AOM board CPLD information of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Reads the CPLD information from an input AOM image file.
    --json_view
          Sets the output format to JSON.
    --file_only
          Works with the --file option, and only reads AOM information from the
          input image file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Multiple Systems Remote In-Band]
    [Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetAomboardCpldInfo --file
        CPLD.bin
  In-Band
    # ./saa -c GetAomboardCpldInfo --file CPLD.bin --file_only
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetAomboardCpldInfo 
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetAomboardCpldInfo --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetAomboardCpldInfo 
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetAomboardCpldInfo
