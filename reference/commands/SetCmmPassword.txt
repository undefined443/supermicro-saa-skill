Description
    Sets CMM user password.
Required Arguments
    None
Optional Arguments
    --user_id    <user ID>
          Enters the CMM user ID.
    --new_password    <new password>
          Sets the new CMM user password.
    --confirm_password    <confirms password>
          Confirms the new CMM user password.
    --pw_file    <Password File>
          The specified file path to read the new CMM user password.
Usage Modes
    [OOB] [Multiple Systems OOB] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c SetCmmPassword --pw_file
        password.txt
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c SetCmmPassword --user_id 3
        --new_password '&123456789' --confirm_password '&123456789'
Notice
    1. If password includes special character(s), it has to be quoted.
    2. Without --user_id, the user ID is set to 2 (the default Administrator
    account).
