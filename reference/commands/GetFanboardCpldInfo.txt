Description
    Gets the Fanboard CPLD information of the managed system.
Required Arguments
    None
Optional Arguments
    None
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetFanboardCpldInfo 
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetFanboardCpldInfo 
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetFanboardCpldInfo 
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetFanboardCpldInfo 
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetFanboardCpldInfo
