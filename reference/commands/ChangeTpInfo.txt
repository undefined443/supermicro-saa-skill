Description
    Updates the TwinPro configuration.
Required Arguments
    --file    <file name>
          Updates the TwinPro configuration with the given configuration file.
Optional Arguments
    --individually
          Updates each TwinPro configuration with corresponding configuration
          file individually.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ChangeTpInfo --file TpCfg.xml
  In-Band
    # ./saa -c ChangeTpInfo --file TpCfg.xml
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        ChangeTpInfo --file TpCfg.xml
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChangeTpInfo --file
        TpCfg.xml
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChangeTpInfo --file
        TpCfg.xml --individually
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c ChangeTpInfo --file TpCfg.xml
Notice
    1. --individually option should be used with -l option for concurrent OOB
    usage.
