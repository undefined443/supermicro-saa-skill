Description
    Gets the AOC NIC information of the managed system/input AOC_NIC image file.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Reads the AOC NIC firmware information from an input AOC_NIC image
          file.
    --dev_id    <Device ID>
          AOC NIC device ID list.
    --file_only
          Works with the --file option, and only reads AOC-NIC information from
          the input image file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetAocNICInfo --dev_id 1
  In-Band
    # ./saa -c GetAocNICInfo --file AOC_NIC.bin --file_only
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetAocNICInfo --dev_id 1,2,3
        --file AOC_NIC.bin
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetAocNICInfo --dev_id 1,2,3 --file AOC_NIC.bin --remote_saa
        /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetAocNICInfo --file
        AOC_NIC.bin
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetAocNICInfo --dev_id 1,2,3
        --file AOC_NIC.bin --remote_saa /root/saa
Notice
    1. Supports only on X12/H12 and later platforms.
