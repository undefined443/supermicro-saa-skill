Description
    Gets the system configuration (BIOS and BMC) of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Saves the configuration to a file.
    --file_id    <file ID>
          Downloads existing Blade system profile from CMM with specific file
          ID.
    --current_password    <current password>
          Checks the current BIOS Administrator password.
    --download
          Downloads current Blade system configuration file that support profile
          update from CMM.
    --overwrite
          Overwrites the output file.
    --dev_id    <Device ID>
          Assigns Blade index and node ID.
          Blade index: [A1-A14] or [B1-B14]
          Node ID: [1-4]
          Format: [A1_1]
    --action    <action>
          Sets the action of each XML table. Acceptable option: None or Change.
    --cur_pw_file    <Current password file>
          The specified file path to read BIOS Administrator password.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetSystemCfg --file
        SystemCfg.xml --overwrite
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetSystemCfg --download
        --dev_id A1_1 --file SystemCfg.xml --overwrite
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetSystemCfg --download
        --file_id 2 --file SystemCfg_Cache.xml --overwrite
  In-Band
    # ./saa -c GetSystemCfg --file SystemCfg.xml --overwrite
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        GetSystemCfg --file SystemCfg.xml --overwrite
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetSystemCfg --file
        SystemCfg.xml --overwrite
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetSystemCfg --download
        --dev_id A1_1 --file SystemCfg.xml --overwrite
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetSystemCfg --download
        --file_id 2 --file SystemCfg_Cache.xml --overwrite
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetSystemCfg --file
        SystemCfg.xml --overwrite
