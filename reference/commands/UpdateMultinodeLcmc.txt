Description
    Updates the multi-node LCMC.
Required Arguments
    --file    <file name>
          Updates the multi-node LCMC with the given multi-node LCMC image file.
Optional Arguments
    None
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c UpdateMultinodeLcmc --file
        LCMC.bin
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c UpdateMultinodeLcmc --file
        LCMC.bin
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c UpdateMultinodeLcmc --file LCMC.bin --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c UpdateMultinodeLcmc --file
        LCMC.bin
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c UpdateMultinodeLcmc --file
        LCMC.bin
Notice
    1. Only can be operated on node A system to update multi-node LCMC.
