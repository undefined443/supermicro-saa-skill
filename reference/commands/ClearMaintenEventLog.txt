Description
    Clears the maintenance event log of the managed system.
Required Arguments
    None
Optional Arguments
    --gen_log
          Generates a log entry indicating the successful clearing of the
          maintenance event log.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ClearMaintenEventLog 
  In-Band
    # ./saa -c ClearMaintenEventLog --gen_log
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        ClearMaintenEventLog --gen_log --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ClearMaintenEventLog 
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c ClearMaintenEventLog
