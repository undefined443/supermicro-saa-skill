Description
    Updates motherboard FPGA.
Required Arguments
    --file    <file name>
          Updates the motherboard FPGA with the given FW image file.
    --reboot
          Forces the managed system to reboot or power up after operation.
Optional Arguments
    --individually
          Updates each motherboard FPGA with corresponding image file
          individually.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c UpdateMotherboardFpga --file
        FPGA.bin --reboot
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c UpdateMotherboardFpga --file
        FPGA.bin --reboot
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c UpdateMotherboardFpga --file FPGA.bin --reboot --remote_saa
        /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c UpdateMotherboardFpga
        --file FPGA.bin --reboot
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c UpdateMotherboardFpga --file
        FPGA.bin --reboot
Notice
    1. System needs to do AC cycle after FPGA update.
