Description
    Gets the BMC LAN configuration of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Saves the configuration to a file.
          Prints the BMC LAN configuration on screen if the file-saving function
          is not available.
    --overwrite
          Overwrites the output file.
    --action    <action>
          Sets the action of each XML table. Acceptable option: None or Change.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetBmcLANCfg --file
        BmcLANCfg.txt --overwrite
  In-Band
    # ./saa -c GetBmcLANCfg --file BmcLANCfg.txt --overwrite
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetBmcLANCfg --file BmcLANCfg.txt
        --overwrite
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        GetBmcLANCfg --file BmcLANCfg.txt --overwrite --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetBmcLANCfg --file BmcLANCfg.txt --overwrite
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetBmcLANCfg --file
        BmcLANCfg.txt --overwrite
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetBmcLANCfg --file
        BmcLANCfg.txt --overwrite
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetBmcLANCfg --file
        BmcLANCfg.txt --overwrite
