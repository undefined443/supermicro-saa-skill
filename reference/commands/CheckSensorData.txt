Description
    Gets the sensor values of the managed system.
Required Arguments
    None
Optional Arguments
    --sdr_id    <SDR id>
          The SDR ID for delete.
    --sdr_major_version    <SDR major version>
          The SDR major version.
    --sdr_minor_version    <SDR minor version>
          The SDR minor version.
    --redfish
          Enables pure Redfish support.
    --showall
          Shows the sensor all threshold values of the managed system.
    --json_view
          Sets the output format to JSON.
    --action    <action>
          Sets action to:
          1 = Show
          2 = Del
          3 = GetVer
          4 = SetVer
    --temp_unit    <TEMP_UNIT>
          Specifies the temperature unit to use. Acceptable values: 'C' for
          Celsius, 'F' for Fahrenheit.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c CheckSensorData 
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c CheckSensorData --showall
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c CheckSensorData --action Show
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c CheckSensorData --action Show
        --showall
  In-Band
    # ./saa -c CheckSensorData --action GetVer
    # ./saa -c CheckSensorData --action Del --sdr_id 1234
    # ./saa -c CheckSensorData --temp_unit C
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c CheckSensorData --action Show
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c CheckSensorData --showall
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c CheckSensorData --action Show
        --showall
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        CheckSensorData --action GetVer --remote_saa /root/saa
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        CheckSensorData --action Del --sdr_id 1234 --remote_saa /root/saa
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        CheckSensorData --temp_unit C --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c CheckSensorData --action Show --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c CheckSensorData --showall --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c CheckSensorData --action Show --showall --remote_saa /root/saa
