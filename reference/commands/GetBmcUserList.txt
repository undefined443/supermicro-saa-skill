Description
    Gets BMC user list.
Required Arguments
    None
Optional Arguments
    --json_view
          Sets the output format to JSON.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Multiple Systems Remote In-Band]
    [Remote In-Band] 
Node Product Key Required
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetBmcUserList 
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetBmcUserList --json_view
  In-Band
    # ./saa -c GetBmcUserList 
    # ./saa -c GetBmcUserList --json_view
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetBmcUserList 
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetBmcUserList --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetBmcUserList 
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetBmcUserList 
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetBmcUserList
