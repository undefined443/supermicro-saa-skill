Description
    Activates the node product key of the managed system.
Required Arguments
    None
Optional Arguments
    --key    <node product key value>
          Uses the node product key to activate the managed system.
    --key_file    <node product key file>
          Uses the file of node product key to activate the managed system.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ActivateProductKey --key
        1111-1111-1111-1111-1111-1111
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ActivateProductKey --key_file
        keyfile.txt
  In-Band
    # ./saa -c ActivateProductKey --key 1111-1111-1111-1111-1111-1111
    # ./saa -c ActivateProductKey --key_file keyfile.txt
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c ActivateProductKey --key_file
        keyfile.txt
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        ActivateProductKey --key_file keyfile.txt --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c ActivateProductKey --key_file keyfile.txt --remote_saa
        /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ActivateProductKey 
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ActivateProductKey
        --key_file keyfile.txt
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c ActivateProductKey --key_file
        keyfile.txt --remote_saa /root/saa
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c ActivateProductKey --key_file
        keyfile.txt --remote_saa /root/saa
Notice
    1. A JSON-style format node product key must be put into single quotation
    marks.
