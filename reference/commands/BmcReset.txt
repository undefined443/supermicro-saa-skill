Description
    Executes command to reset BMC.
Required Arguments
    --action    <action>
          Sets action to:
          1 = ColdReset
          2 = WarmReset
Optional Arguments
    --bmc_boot_check
          Checks BMC boots up within 4 minutes after reset.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c BmcReset --action WarmReset
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c BmcReset --bmc_boot_check
        --action WarmReset
  In-Band
    # ./saa -c BmcReset --action ColdReset
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c BmcReset --action ColdReset
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c BmcReset --remote_saa /root/saa --action WarmReset
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c BmcReset --bmc_boot_check
        --action ColdReset
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c BmcReset --bmc_boot_check
        --action WarmReset
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c BmcReset --action ColdReset
