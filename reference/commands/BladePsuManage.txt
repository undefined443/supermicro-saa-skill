Description
    Manages the power supply unit of managed Blade system through CMM.
Required Arguments
    --action    <action>
          Manages system with action:
          GetBladePsuInfo
          GetBladePsuConsumption
          GetFanSpeed
          SetFanSpeed
          GetFanMode
          SetFanMode
Optional Arguments
    None
Usage Modes
    [OOB] [Multiple Systems OOB] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c BladePsuManage --action
        GetBladePsuInfo -h
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c BladePsuManage --action
        GetBladePsuInfo -h
