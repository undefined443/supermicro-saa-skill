Description
    Gets the BMC information of the managed system/input BMC image file.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Reads the BMC information from an input BMC image file.
    --individually
          Gets each BMC with its corresponding image file individually.
    --redfish
          Enables support for pure Redfish.
    --showall
          Display all information.
    --json_view
          Sets the output format to JSON.
    --file_only
          Works with --file option, and only reads BMC information from the
          input image file.
    --extract_measurement
          Works with the --file option, and extracts BMC image file measurement.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetBmcInfo --file BMC.bin
  In-Band
    # ./saa -c GetBmcInfo --file BMC.bin --file_only
    # ./saa -c GetBmcInfo --file BMC.bin
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetBmcInfo 
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c GetBmcInfo
        --file BMC.bin
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetBmcInfo --file BMC.bin
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetBmcInfo --file BMC.bin
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetBmcInfo --file BMC.bin
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetBmcInfo --file BMC.bin
