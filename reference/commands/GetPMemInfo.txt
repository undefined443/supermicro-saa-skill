Description
    Gets the PMem information of the managed system/input PMem image file.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Reads the PMem information from an input PMem image file.
    --json_view
          Sets the output format to JSON.
    --file_only
          Works with the --file option, and only reads PMem information from the
          input image file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetPMemInfo --file PMem.bin
  In-Band
    # ./saa -c GetPMemInfo --file PMem.bin --file_only
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetPMemInfo --file PMem.bin
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetPMemInfo --file PMem.bin
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetPMemInfo --file
        PMem.bin
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetPMemInfo --file PMem.bin
Notice
    1. -I Redfish_HI is required for in-band usage on X12 and later platforms.
