Description
    Gets the event log of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Saves the event log to a file.
          Prints the event log on screen if the file-saving function is not
          available.
    --overwrite
          Overwrites the output file.
    --redfish
          Enables pure Redfish support.
    --json_view
          Sets the output format to JSON.
    --info
          Print the current and total capacity of event log.
    --raw_data
          Prints the raw data of each event log.
    --mfg
          Prints the general information of event logs for the managed system in
          a style that complies with the manufacturer's assembly line
          requirements.
    --year    <year>
          Filter event logs within n years
    --month    <month>
          Filter event logs within n months
    --day    <day>
          Filter event logs within n days
    --format    <file format>
          Saves the event log to a file in CSV format.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetEventLog 
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetEventLog --info
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetEventLog --raw_data
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetEventLog --mfg
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetEventLog --year 1
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetEventLog --file log
        --format csv --overwrite
