Description
    Clears CMOS and resets all BIOS settings to their default values.
Required Arguments
    --ac_cycle
          Proceeds with an AC cycle to the managed system after the operation.
Optional Arguments
    --post_complete
          Waits for the managed system's POST to complete after AC power cycle.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ClearCMOS --ac_cycle
        --post_complete
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c ClearCMOS --ac_cycle
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c ClearCMOS --ac_cycle
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ClearCMOS --ac_cycle
        --post_complete
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c ClearCMOS --ac_cycle
Notice
    1. Clear CMOS will initiates an AC power cycle. BMC will reset and the
    system will reboot.
