Description
    Gets the NVMe SSD information of the managed system.
Required Arguments
    None
Optional Arguments
    --dev_id    <Device ID>
          NVMe device controller ID.
          Prints all NVMe information on the screen if this option is not
          available.
    --redfish
          Enables pure Redfish support.
    --json_view
          Sets the output format to JSON.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetNvmeInfo  --dev_id 0
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetNvmeInfo 
  In-Band
    # ./saa -c GetNvmeInfo  --dev_id 0
    # ./saa -c GetNvmeInfo 
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetNvmeInfo  --dev_id 0
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetNvmeInfo 
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        GetNvmeInfo --dev_id 0 --remote_saa /root/saa
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        GetNvmeInfo --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetNvmeInfo --dev_id 0 --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetNvmeInfo --remote_saa /root/saa
Multiple Systems

  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetNvmeInfo --dev_id 0
        --remote_saa /root/saa
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetNvmeInfo  --remote_saa
        /root/saa
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetNvmeInfo --dev_id 0
        --remote_saa /root/saa
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetNvmeInfo  --remote_saa
        /root/saa
