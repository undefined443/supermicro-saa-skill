Description
    Gets the AIP (AI Processor) CPLD information of the managed system.
Required Arguments
    None
Optional Arguments
    None
Usage Modes
    [OOB] [Multiple Systems OOB] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetAipCpldInfo 
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetAipCpldInfo 
Notice
    1. Supports for Gaudi HL205 AIP on Habana system only.
