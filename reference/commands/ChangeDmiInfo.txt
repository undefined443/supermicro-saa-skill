Description
    Updates DMI information.
Required Arguments
    --file    <file name>
          Updates the DMI information with the given text file.
Optional Arguments
    --reboot
          Forces the managed system to reboot or power up after operation.
    --individually
          Individually updates each piece of DMI information with its
          corresponding text file
    --post_complete
          Waits for the managed system POST complete after rebooting.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ChangeDmiInfo --file dmi.txt
        --reboot
  In-Band
    # ./saa -c ChangeDmiInfo --file dmi.txt --reboot
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        ChangeDmiInfo --file dmi.txt --reboot --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChangeDmiInfo --file
        dmi.txt --reboot
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChangeDmiInfo --file
        dmi.txt --individually --reboot
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c ChangeDmiInfo --file dmi.txt
        --reboot --remote_saa /root/saa
Notice
    1. --inidvidually option should be used with -l option for multiple OOB
    usage.
