Description
    Downloads the BMC configuration of the managed system.
Required Arguments
    --file    <file name>
          Downloads the BMC configuration to a file.
Optional Arguments
    --overwrite
          Overwrites the output file.
    --format    <file format>
          Works with the --file option to download the BMC configuration to a
          file in one of the following specified formats:
          BINARY = Binary format
          TEXT = Text format
          If the --format option is not provided, the default format is BINARY.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c DownloadBmcCfg --format BINARY
        --file BmcCfg.bin --overwrite
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c DownloadBmcCfg --format TEXT
        --file BmcCfg.xml --overwrite
  In-Band
    # ./saa -c DownloadBmcCfg --format BINARY --file BmcCfg.bin --overwrite
    # ./saa -c DownloadBmcCfg --format TEXT --file BmcCfg.xml --overwrite
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        DownloadBmcCfg --format BINARY --file BmcCfg.bin --overwrite
        --remote_saa /root/saa
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        DownloadBmcCfg --format TEXT --file BmcCfg.xml --overwrite --remote_saa
        /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c DownloadBmcCfg --format
        BINARY --file BmcCfg.bin --overwrite
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c DownloadBmcCfg --format
        TEXT --file BmcCfg.xml --overwrite
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c DownloadBmcCfg --format BINARY
        --file BmcCfg.bin --overwrite --remote_saa /root/saa
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c DownloadBmcCfg --format TEXT
        --file BmcCfg.xml --overwrite --remote_saa /root/saa
