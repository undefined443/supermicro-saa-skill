Description
    Enables a screenVideo remotely.
Required Arguments
    --action    <action>
          Sets action to:
          1 = Download
          2 = Enable
          3 = Disable
Optional Arguments
    --file    <file name>
          Saves the screen recording to a file using the specified name.
    --overwrite
          Overwrites the output file.
    --reboot
          Forces the managed system to reboot or power up after operation.
    --post_complete
          Waits for the managed system's POST to complete after reboot.
    --period    <Period>
          Sets period to:
          1 = BufferFull
          2 = POSTEnd
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c RemoteScreenVideo --action
        Enable --period POSTEnd --reboot --post_complete
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c RemoteScreenVideo --action
        Download --file filename.avi --overwrite
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c RemoteScreenVideo --action
        Disable
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c RemoteScreenVideo --action Enable
        --period POSTEnd --reboot
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c RemoteScreenVideo --action
        Download --file filename.avi --overwrite
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c RemoteScreenVideo --action
        Disable
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c RemoteScreenVideo --action Enable --period POSTEnd --reboot
        --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c RemoteScreenVideo --action Download --file filename.avi
        --overwrite --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c RemoteScreenVideo --action Disable --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c RemoteScreenVideo --action
        Enable --period POSTEnd --reboot --post_complete
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c RemoteScreenVideo --action
        Download --file filename.avi --overwrite
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c RemoteScreenVideo --action
        Disable
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c RemoteScreenVideo --action
        Enable --period POSTEnd --reboot --remote_saa /root/saa
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c RemoteScreenVideo --action
        Download --file filename.avi --overwrite --remote_saa
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c RemoteScreenVideo --action
        Disable --remote_saa /root/saa
Notice
    1. The remote screenVideo is saved in AVI format.
