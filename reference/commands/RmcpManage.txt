Description
    Sets RMCP status of managed system.
Required Arguments
    --action    <action>
          Sets RMCP status with:
          1 = GetInfo
          2 = Enable
          3 = Disable
Optional Arguments
    --port    <port>
          Command optional port(s)
          The format is "RMCP:623" or "623".
          Command optional port(s) list:
          1. RMCP (for RMCP service port)
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c RmcpManage  --action Enable
        --port RMCP:623
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c RmcpManage  --action Enable
        --port 623
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c RmcpManage  --action Enable
        --port RMCP:623
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c RmcpManage  --action Enable
        --port 623
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c RmcpManage  --action Enable --port RMCP:623
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c RmcpManage  --action Enable --port 623
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c RmcpManage  --action
        Enable --port RMCP:623
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c RmcpManage  --action
        Enable --port 623
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c RmcpManage  --action Enable
        --port RMCP:623
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c RmcpManage  --action Enable
        --port 623
