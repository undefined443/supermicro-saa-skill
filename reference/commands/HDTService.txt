Description
    Get/Set HDT(Hardware Debug Tool) service.
Required Arguments
    --action    <action>
          Sets action to:
          1 = GetCurrentStatus
          2 = Enable
          3 = Disable
Optional Arguments
    None
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c HDTService --action
        GetCurrentStatus
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c HDTService --action Enable
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c HDTService --action
        GetCurrentStatus
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c HDTService --action Enable
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c HDTService --action GetCurrentStatus --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c HDTService --action Enable --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c HDTService --action
        GetCurrentStatus
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c HDTService --action
        Disable
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c HDTService --action
        GetCurrentStatus --remote_saa /root/saa
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c HDTService --action Disable
        --remote_saa /root/saa
