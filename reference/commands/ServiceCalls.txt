Description
    Monitors the status of system event log and sensor data record of the
    managed system.
Required Arguments
    --file    <file name>
          Monitors the host with the given XML-style file listing system event
          logs and sensor data records to be monitored.
Optional Arguments
    None
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE and SFT-DCMS-SVC-KEY
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ServiceCalls --file
        current.xml
  In-Band
    # ./saa -c ServiceCalls --file current.xml
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c ServiceCalls --file current.xml
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        ServiceCalls --file current.xml --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c ServiceCalls --file current.xml --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ServiceCalls --file
        current.xml
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c ServiceCalls --file
        current.xml --remote_saa /root/saa
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c ServiceCalls --file
        current.xml --remote_saa /root/saa
