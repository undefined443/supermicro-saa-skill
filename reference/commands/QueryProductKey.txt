Description
    Queries the node product key of the managed system.
Required Arguments
    None
Optional Arguments
    --show_index
          Prints the key index information.
    --showall
          Prints the key index and full key information.
    --json_view
          Sets the output format to JSON.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c QueryProductKey 
  In-Band
    # ./saa -c QueryProductKey 
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c QueryProductKey 
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        QueryProductKey --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c QueryProductKey --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c QueryProductKey 
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c QueryProductKey --remote_saa
        /root/saa
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c QueryProductKey --remote_saa
        /root/saa
