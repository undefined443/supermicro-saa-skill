Description
    Gets and clears the chassis intrusion status for the managed system.
Required Arguments
    --action    <action>
          Sets action to:
          1 = Status
          2 = Clear
Optional Arguments
    None
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ChassisIntrusion --action
        Status
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ChassisIntrusion --action
        Clear
  In-Band
    # ./saa -c ChassisIntrusion --action Status
    # ./saa -c ChassisIntrusion --action Clear
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c ChassisIntrusion --action Clear
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c ChassisIntrusion --action Status
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        ChassisIntrusion --action Status
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c ChassisIntrusion --action Clear
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChassisIntrusion --action
        Status
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChassisIntrusion --action
        Clear
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c ChassisIntrusion --action
        Status
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c ChassisIntrusion --action
        Clear
