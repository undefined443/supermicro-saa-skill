Description
    Gets the motherboard FPGA information of the managed system/input FPGA image
    file.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Reads the motherboard FPGA information from an input FPGA image file.
    --file_only
          Works with the --file option, and only reads motherboard FPGA
          information from the input image file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetMotherboardFpgaInfo --file
        FPGA.bin
  In-Band
    # ./saa -c GetMotherboardFpgaInfo --file FPGA.bin --file_only
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetMotherboardFpgaInfo --file
        FPGA.bin
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetMotherboardFpgaInfo --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetMotherboardFpgaInfo 
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetMotherboardFpgaInfo
