Description
    Gets the FRU information of the managed system/input dumped FRU file.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Saves the dumped FRU data to a file.
    --overwrite
          Overwrites the output file.
    --dev_id    <Device ID>
          Gets more FRUs from CMM.
          FRU ID: [1-19] or "ALL"
          1 = CMM Master
          2 = CMM Middle Plane
          3 = CMM Switch(A1)
          4 = CMM Switch(A2)
          5 = CMM Switch(B1)
          6 = CMM Switch(B2)
          7 = CMM PSU(A1)
          8 = CMM PSU(A2)
          9 = CMM PSU(A3)
          10 = CMM PSU(A4)
          11 = CMM PSU(B1)
          12 = CMM PSU(B2)
          13 = CMM PSU(B3)
          14 = CMM PSU(B4)
          15 = CMM FAN(1)
          16 = CMM FAN(2)
          17 = CMM FAN(3)
          18 = CMM FAN(4)
          19 = CMM Slave
    --redfish
          Enables support for pure Redfish.
    --showall
          Gets all FRU information from managed system.
    --json_view
          Sets the output format to JSON.
    --file_only
          Works with the --file option and only reads FRU information from the
          input dumped FRU binary file.
    --format    <file format>
          Works with the --file and --dump options to download FRU data to file
          in one of the following specified formats:
          BINARY = Binary format
          TEXT = Text format
          If the --format option is not provided, the default format is BINARY.
    --dump
          Works with the --file option and dumps FRU data.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetFruInfo 
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetFruInfo --dev_id 1,12
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetFruInfo --file dumpedFile
        --dump --format TEXT --overwrite
  In-Band
    # ./saa -c GetFruInfo --file dumpedFile --file_only
    # ./saa -c GetFruInfo --file dumpedFile --dump --overwrite
    # ./saa -c GetFruInfo --file dumpedFile --dump --format BINARY --overwrite
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetFruInfo --redfish
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c GetFruInfo
        --file dumpedFile --dump --format TEXT --overwrite --remote_saa
        /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetFruInfo --redfish
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetFruInfo --file
        dumpedFile --dump --format TEXT --overwrite
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetFruInfo --file dumpedFile
        --dump --format BINARY --overwrite
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetFruInfo --redfish
Notice
    1. The FRU version will be displayed if the managed system supports it.
