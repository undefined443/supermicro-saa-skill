Description
    Gets the BIOS information of the managed system/input BIOS image file.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Reads BIOS information from an input BIOS image file.
    --individually
          Gets each BIOS with its corresponding image file individually.
    --redfish
          Enables support for pure Redfish.
    --showall
          Prints the BIOS version, BIOS revision and BIOS OEM FID information.
    --json_view
          Sets the output format to JSON.
    --file_only
          Works with --file, and only reads BIOS information from the input
          image file.
    --extract_measurement
          Works with --file, extract BIOS image file measurement.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetBiosInfo --file BIOS.bin
  In-Band
    # ./saa -c GetBiosInfo --file BIOS.bin --file_only
    # ./saa -c GetBiosInfo --file BIOS.bin --showall
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetBiosInfo --file BIOS.bin
        --showall
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        GetBiosInfo --file BIOS.bin --showall --remote_saa /root/saa
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        GetBiosInfo --file BIOS.bin --file_only --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetBiosInfo --file BIOS.bin
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetBiosInfo --file
        BIOS.bin
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetBiosInfo --file BIOS.bin
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetBiosInfo --file BIOS.bin
