Description
    Takes a screenshot remotely.
Required Arguments
    --file    <file name>
          Saves the screenshot to a file.
Optional Arguments
    --overwrite
          Overwrites the output file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c RemoteScreenshot --file
        screenshot.png
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c RemoteScreenshot --file
        screenshot.png
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c RemoteScreenshot --file
        screenshot.png
Notice
    1. The remote screenshot is saved in PNG format.
