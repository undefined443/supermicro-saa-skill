Description
    Sets BMC user password.
Required Arguments
    None
Optional Arguments
    --user_id    <user ID>
          Enters the BMC user ID.
    --new_password    <new password>
          Sets the new BMC user password.
    --confirm_password    <confirms password>
          Confirms the new BMC user password.
    --pw_file    <Password File>
          The specified file path to read the new BMC user password.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c SetBmcPassword --pw_file
        password.txt
  In-Band
    # ./saa -c SetBmcPassword --user_id 3 --new_password 987654321
        --confirm_password 987654321
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        SetBmcPassword --user_id 3 --new_password 987654321 --confirm_password
        987654321 --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c SetBmcPassword --user_id 3
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c SetBmcPassword --user_id 3
Notice
    1. If password includes special character(s), it has to be quoted.
    2. Without --user_id, the user ID is set to 2 (the default Administrator
    account).
