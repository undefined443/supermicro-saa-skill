Description
    Sets secure boot.
Required Arguments
    --action    <action>
          Sets action to:
          1 = Status
          2 = Enable
          3 = Disable
          4 = ShowDatabases        
          5 = UploadCertificate
          6 = ResetAllKeysToDefault
          7 = DeleteAllKeys
          8 = DeletePK
Optional Arguments
    --file    <file name>
          Uploads secure boot key in the format of PEM.
    --file_type    <file type>
          Selects the type of secure boot key.The format of <file type> is
          "PK","KEK", "db", "dbr", "dbt" or "dbx" (case sensitive).
          Note that "dbx" can be only used with the "ShowDatabases" action
    --reboot
          Forces the managed system to reboot or power up after operation.
    --individually
          Updates each system secure boot keys individually with the
          corresponding secure boot key.
    --post_complete
          Waits for the managed system's POST to complete after reboot.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c SecureBootManage --action
        Status
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c SecureBootManage --action
        Enable
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c SecureBootManage --action
        DeletePK
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c SecureBootManage --action Status
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c SecureBootManage --action Enable
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c SecureBootManage --action ShowDatabases
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c SecureBootManage --action
        UploadCertificate --file_type PK --file certificate
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c SecureBootManage --action
        ResetAllKeysToDefault
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c SecureBootManage --action
        DeleteAllKeys
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c SecureBootManage --action
        Disable
