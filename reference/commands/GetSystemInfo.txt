Description
    Obtains a summary of information from the managed system.
Required Arguments
    None
Optional Arguments
    --redfish
          Enables support for pure Redfish.
    --json_view
          Sets the output format to JSON.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetSystemInfo 
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetSystemInfo --redfish
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetSystemInfo --json_view
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetSystemInfo --redfish
        --json_view
  In-Band
    # ./saa -c GetSystemInfo 
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetSystemInfo 
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        GetSystemInfo --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetSystemInfo --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetSystemInfo 
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetSystemInfo 
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetSystemInfo 
Notice
    1. For NVIDIA systems, the "GetSystemInfo" command only supports the Redfish
    protocol. For in-band access, please use the Redfish host interface with the
    "-I Redfish_HI" option.
