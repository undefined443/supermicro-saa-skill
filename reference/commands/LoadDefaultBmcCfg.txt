Description
    Loads the default factory BMC configuration.
Required Arguments
    None
Optional Arguments
    --reboot
          Forces the managed system to reboot or power up after operation.
    --redfish
          Enables support for pure Redfish.
    --clear_user_cfg
          Clears user configuration.
    --preserve_user_cfg
          Preserves user configuration.
    --load_unique_password
          Loads the unique BMC password.
    --load_default_password
          Loads the default BMC password.
    --load_default_lan
          Loads the default BMC LAN configuration.
    --load_default_fru
          Loads the default FRU configuration.
    --reset_all
          Loads default configuration for NVIDIA platforms.
          Different NVIDIA platforms may load different configurations depending
          on platform capabilities as following:
          - NVIDIA C2: Loads default BMC and BIOS configurations.
    --post_complete
          Waits for the managed system POST complete after rebooting.
    --bmc_boot_check
          Checks if BMC is booted up after reset.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c LoadDefaultBmcCfg
        --preserve_user_cfg --bmc_boot_check --reboot --post_complete
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c LoadDefaultBmcCfg
        --clear_user_cfg --load_unique_password
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c LoadDefaultBmcCfg
        --clear_user_cfg --load_default_password
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c LoadDefaultBmcCfg
        --clear_user_cfg --load_default_lan --load_unique_password
        --load_default_fru
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c LoadDefaultBmcCfg --reset_all
        --bmc_boot_check --reboot
  In-Band
    # ./saa -c LoadDefaultBmcCfg --preserve_user_cfg --bmc_boot_check --reboot
    # ./saa -c LoadDefaultBmcCfg --clear_user_cfg --load_unique_password
        --bmc_boot_check --reboot
    # ./saa -c LoadDefaultBmcCfg --clear_user_cfg --load_default_password
        --bmc_boot_check --reboot
    # ./saa -c LoadDefaultBmcCfg --clear_user_cfg --load_default_lan
        --load_unique_password --bmc_boot_check --reboot
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c LoadDefaultBmcCfg
        --preserve_user_cfg
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c LoadDefaultBmcCfg --reset_all
        --reboot
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        LoadDefaultBmcCfg --preserve_user_cfg --bmc_boot_check --reboot
        --remote_saa /root/saa
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        LoadDefaultBmcCfg --clear_user_cfg --load_unique_password
        --bmc_boot_check --reboot
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        LoadDefaultBmcCfg --clear_user_cfg --load_default_password
        --bmc_boot_check --reboot
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c LoadDefaultBmcCfg --preserve_user_cfg

        --clear_user_cfg --load_unique_password
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c LoadDefaultBmcCfg
        --clear_user_cfg --load_default_password
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c LoadDefaultBmcCfg
        --preserve_user_cfg
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c LoadDefaultBmcCfg
        --clear_user_cfg --load_unique_password
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c LoadDefaultBmcCfg
        --clear_user_cfg --load_default_password
Notice
    1. The option [--clear_user_cfg] has to pair with option
    [--load_unique_password] or [--load_default_password].
    2. The option [--load_default_lan] has to pair with options
    [--clear_user_cfg] and one of [--load_default_password] or
    [--load_unique_password].
    3. The option [--load_default_fru] has to pair with options
    [--clear_user_cfg], [--load_default_lan] and one of
    [--load_default_password] or [--load_unique_password].
    4. The option [--reboot] has to pair with option [--bmc_boot_check].
    5. The option [--reset_all] is only supported for NVIDIA platform.
