Description
    Controls the BMC Redfish session information of the managed system.
Required Arguments
    --action    <action>
          Sets action to:
          1 = GetInfo
          2 = Disconnect
Optional Arguments
    --session_id    <SESSION ID>
          Specifies the session with the given session ID.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c SessionManage --action GetInfo
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c SessionManage --action Disconnect
        --session_id 1
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c SessionManage --action GetInfo --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c SessionManage --action
        GetInfo
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c SessionManage --action GetInfo
