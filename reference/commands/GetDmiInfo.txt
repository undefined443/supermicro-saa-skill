Description
    Gets the DMI information of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Saves the DMI information to a file.
          Prints the DMI information on the screen if the file-saving function
          is not available.
    --overwrite
          Overwrites the output file.
    --json_view
          Sets the output format to JSON.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetDmiInfo --file dmi.txt
        --overwrite
  In-Band
    # ./saa -c GetDmiInfo --file dmi.txt --overwrite
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c GetDmiInfo
        --file dmi.txt --overwrite --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetDmiInfo --file dmi.txt
        --overwrite
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetDmiInfo --file dmi.txt
        --overwrite --remote_saa /root/saa
