Description
    Manages the CPU of managed system.
Required Arguments
    --type    <type>
          Manages CPU with type:
          NM20
          NM40
Optional Arguments
    None
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c NmCpuManage --type NM40 -h
  In-Band
    # ./saa -c NmCpuManage --type NM40 -h
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c NmCpuManage --type NM40 -h
