Description
    Sets BIOS Administrator password.
Required Arguments
    None
Optional Arguments
    --new_password    <new password>
          Sets the new BIOS Administrator password.
    --confirm_password    <confirms password>
          Confirms the new BIOS Administrator password.
    --current_password    <current password>
          Checks the current BIOS Administrator password.
    --reboot
          Forces the managed system to reboot or power up after operation.
    --pw_file    <Password File>
          The specified file path to read new password.
    --post_complete
          Waits for the managed system POST complete after rebooting.
    --cur_pw_file    <Current password file>
          The specified file path to read current password.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c SetBiosPassword
        --current_password '' --new_password '' --confirm_password ''
  In-Band
    # ./saa -c SetBiosPassword --current_password 123456 --new_password 654321
        --confirm_password 654321 --reboot
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        SetBiosPassword --current_password 123456 --new_password 654321
        --confirm_password 654321 --reboot --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c SetBiosPassword --reboot
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c SetBiosPassword --reboot
Notice
    1. Either [--new_password and --confirm_password] or pw_file option is
    applied.
    2. If password includes special character(s), it has to be quoted.
    3. Quoted empty string will clear BIOS Administrator password.
    4. Either [--current_password] or cur_pw_file option is required when BIOS
    Administrator password has been set.
    5. The multi-OOB machine list supports the following two formats:
    [IP_or_HostName NewBiosPassword CurrentBiosPassword]
    [IP_or_HostName BMC_Username BMC_Password NewBiosPassword
    CurrentBiosPassword]
    * CurrentBiosPassword is REQUIRED for these two formats. Please leave a
    space character if BIOS password is not installed.
