Description
    Diagnoses the AMD MI250 GPU status of the managed system.
Required Arguments
    None
Optional Arguments
    --dev_id    <Device ID>
          The device ID list of GPU on the managed system.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c DiagGpuStatus --dev_id 1,2,3,4
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c DiagGpuStatus 
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c DiagGpuStatus --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c DiagGpuStatus --dev_id
        5,6,7,8
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c DiagGpuStatus --remote_saa
        /root/saa
Notice
    1. If --dev_id is not applied, the default will print all GPU devices on the
    managed system.
    2. This function supports mainly for diagnosing AMD MI250 GPU platform.
