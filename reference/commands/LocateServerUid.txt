Description
    Executes command to control UID.
Required Arguments
    --action    <action>
          Sets action to:
          1 = GetStatus
          2 = On
          3 = Off
Optional Arguments
    --json_view
          Sets the output format to JSON.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c LocateServerUid --action 3
  In-Band
    # ./saa -c LocateServerUid --action On
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c LocateServerUid --action 2
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c LocateServerUid --action
        GetStatus
