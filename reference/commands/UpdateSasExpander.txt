Description
    Updates SAS Expander.
Required Arguments
    --file    <file name>
          Updates SAS Expander with the given image file.
    --index    <index>
          Set the SAS Expander index that needs to be updated.
Optional Arguments
    --reboot
          Forces the managed system to power cycle after operation.
Usage Modes
    [OOB] [In-Band] [Remote In-Band] [Multiple Systems Remote In-Band] 
Node Product Key Required
    No
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c UpdateSasExpander --file
        SAS_fw.rom --reboot --index 1
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c UpdateSasExpander --file
        SAS_fw.rom--index 1
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c UpdateSasExpander --file SAS_fw.rom --reboot --index 1
        --remote_saa /root/saa
Multiple Systems
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c UpdateSasExpander --file
        SAS_fw.rom --reboot --index 1
Notice
    1. System needs to be power cycle after updating SAS Expander FW in order to
    take effect.
