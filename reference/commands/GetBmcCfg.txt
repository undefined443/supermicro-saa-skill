Description
    Gets the BMC configuration of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Saves the configuration to a file.
          Prints the BMC configuration on screen if the file-saving function is
          not available.
    --sample_file    <sample file>
          Creates BMC configuration using the format from the sample file.
    --overwrite
          Overwrites the output file.
    --action    <action>
          Sets the action of each XML table. Acceptable option: None or Change.
    --dump
          Dumps read-only BMC configuration file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetBmcCfg --file BmcCfg.xml
        --overwrite
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetBmcCfg --dump --file
        BmcCfg.bin --overwrite
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetBmcCfg --file BmcCfg.xml
        --overwrite --sample_file config_format.xml
  In-Band
    # ./saa -c GetBmcCfg --file BmcCfg.xml --overwrite
    # ./saa -c GetBmcCfg --file BmcCfg.xml --overwrite --sample_file
        config_format.xml
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetBmcCfg --file BmcCfg.xml

    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetBmcCfg --dump --file

    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetBmcCfg --file BmcCfg.xml

        --file BmcCfg.xml --overwrite --remote_saa /root/saa

        --file BmcCfg.xml --overwrite --sample_file config_format.xml

    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
