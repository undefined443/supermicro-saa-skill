Description
    Executes RoT-related actions.
Required Arguments
    --action    <action>
          Sets action to:
          1 = GetInfo
          2 = UpdateGolden
Optional Arguments
    None
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c CpldRotManage --action GetInfo
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c CpldRotManage --action GetInfo
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c CpldRotManage --action
        UpdateGolden
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c CpldRotManage --action GetInfo --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c CpldRotManage --action
        GetInfo
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c CpldRotManage --action GetInfo
Notice
    1. Support only on RoT system of X13/H13 and later platforms.
    2. UpdateGolden action will overwrite golden image with current CPLD
    firmware.
