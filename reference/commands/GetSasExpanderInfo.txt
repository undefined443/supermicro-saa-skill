Description
    Gets the SAS Expander information of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Reads the SAS Expander information from an input image file.
    --json_view
          Sets the output format to JSON.
    --file_only
          Works with the --file option, and only reads SAS Expander information
          from the input image file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetSasExpanderInfo 
  In-Band
    # ./saa -c GetSasExpanderInfo --file SAS_fw.rom --file_only
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetSasExpanderInfo --json_view
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetSasExpanderInfo --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetSasExpanderInfo 
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetSasExpanderInfo
