Description
    Gets the CPLD information of the managed system/input CPLD image file.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Reads the CPLD information from an input CPLD image file.
    --individually
          Gets each CPLD with its corresponding image file individually.
    --json_view
          Sets the output format to JSON.
    --file_only
          Works with the --file option, and only reads CPLD information from the
          input image file.
    --extract_measurement
          Works with the --file option, extract CPLD image file measurement.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetCpldInfo --file CPLD.bin
  In-Band
    # ./saa -c GetCpldInfo -I Redfish_HI -u ADMIN -p ADMIN
    # ./saa -c GetCpldInfo --file CPLD.bin --file_only
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetCpldInfo --file CPLD.bin
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetCpldInfo --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetCpldInfo --file CPLD.bin --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetCpldInfo --file
        CPLD.bin
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetCpldInfo 
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetCpldInfo --file CPLD.bin
Notice
    1. Support only on RoT system of X12/H12 and later platforms.
    2. -I Redfish_HI is required for in-band usage on X12 and later platforms.
