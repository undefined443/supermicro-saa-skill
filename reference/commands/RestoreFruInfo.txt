Description
    Restores dumped FRU info to the managed system.
Required Arguments
    --file    <file name>
          Reads dumped FRU file.
Optional Arguments
    --individually
          Restores each BMC with corresponding FRU info file individually.
    --format    <file format>
          Works with the --fileoption Works with the --file option to read a FRU
          file inone of the following specified formats:
          BINARY = Binary format
          TEXT = Text format
          If the option --format is not provided, the default format is BINARY.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c RestoreFruInfo --file
        dumpedFile --format TEXT
  In-Band
    # ./saa -c RestoreFruInfo --file dumpedFile
    # ./saa -c RestoreFruInfo --file dumpedFile --format BINARY
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        RestoreFruInfo --file dumpedFile --format BINARY --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c RestoreFruInfo --file
        dumpedFile --format BINARY
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c RestoreFruInfo --file
        dumpedFile --format TEXT --individually
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c RestoreFruInfo --file
        dumpedFile --format TEXT --individually --remote_saa /root/saa
