Description
    Gets the current BIOS configuration of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Saves the BIOS configuration to a file.
          Prints the current BIOS configuration on the screen if the file-saving
          function is not available.
    --current_password    <current password>
          Checks the current BIOS Administrator password.
    --overwrite
          Overwrites the output file.
    --tui
          Edits BIOS configuration in text-based user interface.
    --compact
          Generates a compact version of the BIOS configuration containing only
          the settings that have been changed in the text-based user interface.
    --cur_pw_file    <Current password file>
          The specified file path to read current password.
    --filter    <Filter type>
          Sets filter type to:
          1 = nondefault
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetCurrentBiosCfg --file
        current.file --overwrite
  In-Band
    # ./saa -c GetCurrentBiosCfg --file current.file --overwrite
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        GetCurrentBiosCfg --file current.file --overwrite --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetCurrentBiosCfg
        --overwrite
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetCurrentBiosCfg --overwrite
