Description
    Gets the VROC configuration of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Saves the configuration to a file.
          Prints the VROC configuration on screen if the file-saving function is
          not available.
    --overwrite
          Overwrites the output file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetVROCCfg --file VROC.cfg.xml
        --overwrite
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetVROCCfg --file VROC.cfg.xml
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetVROCCfg --file VROC.cfg.xml
        --overwrite
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetVROCCfg --file VROC.cfg.xml --overwrite
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetVROCCfg --file
        VROC.cfg.xml --overwrite
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetVROCCfg --file VROC.cfg.xml
