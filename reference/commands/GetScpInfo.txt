Description
    Gets the SCP information of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Reads SCP information from an input SCP image file.
    --json_view
          Sets the output format to JSON.
    --file_only
          Works with --file, and only reads SCP information from the input image
          file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetScpInfo 
  In-Band
    # ./saa -c GetScpInfo -I Redfish_HI -u ADMIN -p ADMIN
    # ./saa -c GetScpInfo -I Redfish_HI -u ADMIN -p ADMIN --file SCP.tar
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetScpInfo --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetScpInfo --file SCP.tar --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetScpInfo 
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetScpInfo 
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetScpInfo --file SCP.tar
