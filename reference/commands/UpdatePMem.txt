Description
    Updates PMem firmware.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Updates PMem with the given PMem firmware file.
    --current_password    <current password>
          Checks the current BIOS Administrator password.
    --reboot
          Forces the managed system to reboot or power up after operation.
    --restore_default_fw
          Updates the PMem with BIOS built-in PMem firmware.
    --post_complete
          Waits for the managed system POST complete after rebooting.
    --cur_pw_file    <Current password file>
          The specified file path to read current password.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c UpdatePMem --file PMem.bin
        --reboot
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c UpdatePMem
        --restore_default_fw --reboot
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c UpdatePMem --file PMem.bin
        --reboot
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c UpdatePMem
        --restore_default_fw --reboot
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c UpdatePMem --file PMem.bin --reboot
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c UpdatePMem --file PMem.bin
        --reboot
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c UpdatePMem
        --restore_default_fw --reboot
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c UpdatePMem --file PMem.bin
        --reboot
Notice
    1. -I Redfish_HI is required for in-band usage on X12 and later platforms.
