Description
    Gets/Sets the BMC host name.
Required Arguments
    --action    <action>
          Sets BmcHostName action to:
          1 = Get
          2 = Set
Optional Arguments
    --json_view
          Sets the output format to JSON.
    --hostname    <Host Name>
          BMC Host name
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c BmcHostName --action Get
  In-Band
    # ./saa -c BmcHostName --action Set --hostname testHostName
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        BmcHostName --action Set --hostname testHostName --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c BmcHostName --action Get
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c BmcHostName --action Get
