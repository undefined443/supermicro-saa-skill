Description
    Gets the TwinPro information of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Saves the information to a file.
          Prints the TwinPro information on screen if the file-saving function
          is not available.
    --overwrite
          Overwrites the output file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetTpInfo --file TpCfg.xml
        --overwrite
  In-Band
    # ./saa -c GetTpInfo --file TpCfg.xml --overwrite
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c GetTpInfo
        --file TpCfg.xml --overwrite
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetTpInfo --file TpCfg.xml
        --overwrite
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetTpInfo --file TpCfg.xml
        --overwrite
