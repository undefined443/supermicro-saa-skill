Description
    Gets the crash dump of the managed system. Note: the crash dump file is
    compressed.
Required Arguments
    --action    <action>
          Sets action to:
          1 = CreateDump
          2 = DeleteDump
          3 = DirectDump
Optional Arguments
    --file    <file name>
          Saves the crash dump to a file.
    --overwrite
          Overwrites the output file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetHostDump --action 1 --file
        log.tgz
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetHostDump --action 2
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetHostDump --action 3 --file
        log.tgz
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetHostDump --action 1 --file
        log.tgz
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetHostDump --action 3 --file log.tgz --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetHostDump --action
        CreateDump --file log.tgz --overwrite
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetHostDump --action 2
