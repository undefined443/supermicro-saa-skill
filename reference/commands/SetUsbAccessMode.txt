Description
    Sets current USB access mode.
Required Arguments
    --panel    <front/rear>
          The panel to be set
Optional Arguments
    --enable
          Dynamically enables the USB ports in the assigned panel.
    --disable
          Dynamically disables the USB ports in the assigned panel.
Usage Modes
    [In-Band] [Remote In-Band] [Multiple Systems Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  In-Band
    # ./saa -c SetUsbAccessMode --panel rear --disable
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        SetUsbAccessMode --panel front --enable
Multiple Systems
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c SetUsbAccessMode --panel front
        --disable
