Description
    Prints the PSU information on the managed system.
Required Arguments
    None
Optional Arguments
    --json_view
          Sets the output format to JSON.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetPsuInfo  
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetPsuInfo --json_view
  In-Band
    # ./saa -c GetPsuInfo  
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetPsuInfo  
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c GetPsuInfo

    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetPsuInfo  
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetPsuInfo  
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetPsuInfo  
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetPsuInfo  
Notice
    1. -I Redfish_HI is required for in-band usage for X14/H14 and later
    platforms.
