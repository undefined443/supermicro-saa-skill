Description
    Unmounts the ISO image resources.
Required Arguments
    None
Optional Arguments
    --redfish
          Enables support for pure Redfish.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c UnmountIsoImage 
  In-Band
    # ./saa -c UnmountIsoImage 
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        UnmountIsoImage 
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c UnmountIsoImage 
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c UnmountIsoImage 
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c UnmountIsoImage 
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c UnmountIsoImage
