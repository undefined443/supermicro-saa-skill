Description
    Gets the GPU information of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Reads the GPU information from an input GPU image file.
    --showall
          Prints the FRU information on the GPU baseboard of the managed system.
          (Only supports on X11/H11 with HGX2 system and Intel Gaudi 2/3
          system.)
    --file_only
          Works with --file, and only reads GPU information from the input image
          file.
    --showoam    <Display Selected OAM>
          For Intel PVC and Intel Gaudi2/3 system shows individual OAM.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetGpuInfo 
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetGpuInfo --file GPU.bin
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetGpuInfo --showoam 3
  In-Band
    # ./saa -c GetGpuInfo 
    # ./saa -c GetGpuInfo --file GPU.bin
    # ./saa -c GetGpuInfo --file GPU.bin --file_only
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetGpuInfo 
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c GetGpuInfo
        --remote_saa /root/saa
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c GetGpuInfo
        --file GPU.bin --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetGpuInfo --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetGpuInfo 
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetGpuInfo --file GPU.bin
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetGpuInfo --remote_saa
        /root/saa
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetGpuInfo --file GPU.bin
        --remote_saa /root/saa
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetGpuInfo --remote_saa
        /root/saa
Notice
    1. -I Redfish_HI is required for in-band usage on X12/H12 and later
    platforms.
    2. In-band usage is only supported on X12/H12 and later platforms.
