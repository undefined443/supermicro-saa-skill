Description
    Gets the multi-node EC information of the managed system/input multi-node EC
    image file.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Reads the multi-node EC information from an input multi-node EC image
          file.
    --json_view
          Sets the output format to JSON.
    --file_only
          Works with the --file option, and only reads multi-node EC information
          from the input image file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetMultinodeEcInfo --file
        EC.bin
  In-Band
    # ./saa -c GetMultinodeEcInfo --file EC.bin --file_only
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetMultinodeEcInfo --file EC.bin
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetMultinodeEcInfo --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetMultinodeEcInfo --file
        EC.bin
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetMultinodeEcInfo
