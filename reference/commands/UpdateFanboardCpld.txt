Description
    Executes update Fanboard CPLD based on type selected.
Required Arguments
    --file    <file name>
          Updates the Fanboard CPLD with the given Fanboard CPLD image file.
    --type    <type>
          Sets action to:
          1 = Front
          2 = Rear
          or the corresponding Fanboard ID number.
Optional Arguments
    --individually
          Updates each Fanboard CPLD with the corresponding configuration file
          individually.
    --index    <index>
          Sets the CPLD index. The default value is 1, and the index count
          starts from 1.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c UpdateFanboardCpld --type
        Front --file Fanboard_CPLD.bin
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c UpdateFanboardCpld --type Rear
        --file Fanboard_CPLD.bin
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c UpdateFanboardCpld --type Front --file Fanboard_CPLD.bin
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c UpdateFanboardCpld --type
        3 --file Fanboard_CPLD.bin
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c UpdateFanboardCpld --type
        Front --file Fanboard_CPLD.bin
