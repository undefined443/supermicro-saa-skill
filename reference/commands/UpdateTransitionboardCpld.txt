Description
    Update Transitionboard CPLD.
Required Arguments
    --file    <file name>
          Updates the Transitionboard CPLD with the given CPLD image file.
Optional Arguments
    --individually
          Updates each Transitionboard CPLD with corresponding image file
          individually.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c UpdateTransitionboardCpld
        --file Transitionboard_CPLD.bin
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c UpdateTransitionboardCpld --file
        Transitionboard_CPLD.bin
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c UpdateTransitionboardCpld --file Transitionboard_CPLD.bin
        --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c UpdateTransitionboardCpld
        --file Transitionboard_CPLD.bin
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c UpdateTransitionboardCpld
        --file Transitionboard_CPLD.bin
