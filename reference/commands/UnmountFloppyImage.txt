Description
    Unmounts the floppy image file of the managed system.
Required Arguments
    None
Optional Arguments
    None
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c UnmountFloppyImage 
  In-Band
    # ./saa -c UnmountFloppyImage 
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        UnmountFloppyImage --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c UnmountFloppyImage 
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c UnmountFloppyImage
        --remote_saa /root/saa
