Description
    Sets WatchDog timer.
Required Arguments
    --action    <action>
          Sets WatchDog actions to:
          0 = Set
          1 = Info
          2 = Reset
Optional Arguments
    --timer_action    <Watchdog timer actions>
          Sets WatchDog timer actions to:
          0 = NoAction
          1 = HardReset
          2 = PowerDown
          3 = PowerCycle
    --interval    <time interval>
          Sets WatchDog pre-timeout interval in seconds
    --countdown    <Watchdog count down>
          Sets WatchDog initial count down value
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c BmcWatchDog --action set
        --timer_action 1 --interval 10 --countdown 60
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c BmcWatchDog --action info
  In-Band
    # ./saa -c BmcWatchDog --action set --timer_action 0 --interval 10
        --countdown 60
    # ./saa -c BmcWatchDog --action set --timer_action 3 --interval 10
        --countdown 60
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        BmcWatchDog --action info
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c BmcWatchDog --action set
        --timer_action 2 --interval 10 --countdown 60
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c BmcWatchDog --action reset
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c BmcWatchDog --action set
        --timer_action 0 --interval 10 --countdown 60
Notice
    1. with CountDown option value must be set in 0 to 6553 range.
    2. with Interval option value must be set in 0 to 255 range.
    3. Please set the Interval no closer than 3 counts.
