Description
    Updates the RAID configuration.
Required Arguments
    --file    <file name>
          Updates the RAID with the given configuration file.
Optional Arguments
    --individually
          Updates each RAID with corresponding configuration file individually.
    --controller    <Controller>
          <Broadcom/Marvell> Vendor of RAID controller.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ChangeRaidCfg --file RAID.cfg
  In-Band
    # ./saa -c ChangeRaidCfg --file RAID.cfg
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        ChangeRaidCfg --file RAID.cfg
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChangeRaidCfg --file
        RAID.cfg
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChangeRaidCfg --file
        RAID.cfg --individually
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c ChangeRaidCfg --file RAID.cfg
Notice
    1. --individually option should be used with -l option for multiple OOB
    usage.
