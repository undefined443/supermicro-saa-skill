Description
    Gets TPM information of the managed system.
Required Arguments
    None
Optional Arguments
    --current_password    <current password>
          Checks the current BIOS Administrator password.
    --showall
          Prints the NV data and the capability flags (if applicable) of the
          trusted platform module. (Only supported for Intel platform)
    --cur_pw_file    <Current password file>
          The specified file path to read current password.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetTpmInfo --showall
  In-Band
    # ./saa -c GetTpmInfo --showall
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c GetTpmInfo
        --showall
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetTpmInfo --showall
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c GetTpmInfo --showall
