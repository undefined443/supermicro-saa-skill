Description
    Updates BMC LAN configuration.
Required Arguments
    --file    <file name>
          Updates the BMC with the given LAN configuration file.
Optional Arguments
    --individually
          Updates each BMC with corresponding LAN configuration file
          individually.
    --skip_unknown
          Skips the unknown tables or settings in the BMC LAN configuration
          file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ChangeBmcLANCfg --file
        BmcLANCfg.txt
  In-Band
    # ./saa -c ChangeBmcLANCfg --file BmcLANCfg.txt
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c ChangeBmcLANCfg --file
        BmcLANCfg.txt
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        ChangeBmcLANCfg --file BmcLANCfg.txt --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c ChangeBmcLANCfg --file BmcLANCfg.txt
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChangeBmcLANCfg --file
        BmcLANCfg.txt
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChangeBmcLANCfg --file
        BmcLANCfg.txt --individually
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c ChangeBmcLANCfg --file
        BmcLANCfg.txt --individually
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c ChangeBmcLANCfg --file
        BmcLANCfg.txt
Notice
    1. --inidvidually option should be used with -l option for multiple OOB
    usage.
