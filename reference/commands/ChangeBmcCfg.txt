Description
    Updates BMC configuration.
Required Arguments
    --file    <file name>
          Updates the BMC with the given configuration file.
Optional Arguments
    --individually
          Updates each BMC with corresponding configuration file individually.
    --skip_unknown
          Skips the unknown tables or settings in the BMC configuration file.
    --restore
          Restores the BMC configuration with the corresponding read-only
          configuration file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    Yes
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ChangeBmcCfg --file BmcCfg.xml
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c ChangeBmcCfg --restore --file
        BmcCfg.bin
  In-Band
    # ./saa -c ChangeBmcCfg --file BmcCfg.xml
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c ChangeBmcCfg --file BmcCfg.xml
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c ChangeBmcCfg --restore --file
        BmcCfg.bin
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        ChangeBmcCfg --file BmcCfg.xml --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c ChangeBmcCfg --file BmcCfg.xml --remote_saa /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c ChangeBmcCfg --restore --file BmcCfg.bin --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChangeBmcCfg --file
        BmcCfg.xml
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChangeBmcCfg --file
        BmcCfg.xml --individually
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c ChangeBmcCfg --restore
        --file BmcCfg.bin --individually
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c ChangeBmcCfg --file BmcCfg.xml
        --individually --remote_saa /root/saa
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c ChangeBmcCfg --file BmcCfg.xml
        --remote_saa /root/saa
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c ChangeBmcCfg --restore --file
        BmcCfg.bin --individually --remote_saa /root/saa
Notice
    1. --individually option should be used with -l option for multiple OOB
    usage.
