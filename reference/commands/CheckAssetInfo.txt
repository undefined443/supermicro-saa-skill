Description
    Gets the asset information of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Saves the asset information to a file.
          Prints the asset information on screen if the file-saving function is
          not available.
    --overwrite
          Overwrites the output file.
    --json_view
          Sets the output format to JSON.
Usage Modes
    [OOB] [Multiple Systems OOB] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c CheckAssetInfo --file log.txt
        --overwrite
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c CheckAssetInfo
