Description
    Gets/Sets the CPU power limit of the managed system.
Required Arguments
    --value    <assignment value>
          Assigns a power limit value.
    --action    <action>
          Sets action with:
          1 = GetPowerLimit
          2 = SetPowerLimitEnabled
          3 = SetPowerLimitDisabled
Optional Arguments
    --redfish
          Enables support for pure Redfish.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c CpuPowerManage --action
        GetPowerLimit
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c CpuPowerManage --action
        SetPowerLimitEnabled --value 700
  In-Band
    # ./saa -c CpuPowerManage --action GetPowerLimit
    # ./saa -c CpuPowerManage --action SetPowerLimitEnabled --value 700
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c CpuPowerManage --action
        GetPowerLimit
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c CpuPowerManage --action
        SetPowerLimitEnabled --value 700
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c CpuPowerManage --action
        SetPowerLimitDisabled
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        CpuPowerManage --action GetPowerLimit --remote_saa /root/saa
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c
        CpuPowerManage --action SetPowerLimitEnabled --value 700 --remote_saa
        /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c CpuPowerManage --action
        GetPowerLimit
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c CpuPowerManage --action
        SetPowerLimitEnabled --value 700
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c CpuPowerManage --action
        GetPowerLimit --remote_saa /root/saa
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c CpuPowerManage --action
        SetPowerLimitEnabled --value 700 --remote_saa /root/saa
Notice
    1. The ARM and AMD platform support IPMI protocol.
    2. The H14/X14/B14 and after platform support Redfish protocol.
