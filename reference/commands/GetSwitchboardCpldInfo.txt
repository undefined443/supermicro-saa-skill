Description
    Gets the Switchboard CPLD information of the managed system.
Required Arguments
    None
Optional Arguments
    --file    <file name>
          Reads the Switchboard CPLD information from an input image file.
    --file_only
          Works with the --file option, and only reads Switchboard CPLD
          information from the input image file.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GetSwitchboardCpldInfo --file
        Switchboard_CPLD.bin
  In-Band
    # ./saa -c GetSwitchboardCpldInfo --file Switchboard_CPLD.bin --file_only
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GetSwitchboardCpldInfo 
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GetSwitchboardCpldInfo --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GetSwitchboardCpldInfo 
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GetSwitchboardCpldInfo 
Notice
    1. Support only on RoT system of X13/H13 and later platforms.
