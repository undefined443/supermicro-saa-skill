Description
    Gets/Sets the GPU power limit of the managed system.
Required Arguments
    --value    <assignment value>
          Assigns power limit value.
    --item    <item name>
          Item type of GPU:
          1 = HGX
    --dev_id    <Device ID>
          GPU SXM Device ID can consist with multiple numbers, ranging from 1 to
          8 (separate by comma), or All.
    --action    <action>
          Sets action with:
          1 = GetPowerLimit
          2 = SetPowerLimit
Optional Arguments
    None
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GpuPowerManage --item HGX
        --action GetPowerLimit
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c GpuPowerManage --item HGX
        --action SetPowerLimit --value 700 --dev_id all
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GpuPowerManage --item HGX
        --action GetPowerLimit
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c GpuPowerManage --item HGX
        --action SetPowerLimit --value 700 --dev_id 1,2,3
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GpuPowerManage --item HGX --action GetPowerLimit --remote_saa
        /root/saa
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c GpuPowerManage --item HGX --action SetPowerLimit --value 700
        --dev_id 1,2,3 --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GpuPowerManage --item HGX
        --action GetPowerLimit
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c GpuPowerManage --item HGX
        --action SetPowerLimit --value 700 --dev_id 1,2,3
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GpuPowerManage --item HGX
        --action GetPowerLimit --remote_saa /root/saa
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c GpuPowerManage --item HGX
        --action SetPowerLimit --value 700 --dev_id 1,2,3 --remote_saa /root/saa
