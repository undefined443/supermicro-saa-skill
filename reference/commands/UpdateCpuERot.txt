Description
    Updates CPU ERoT.
Required Arguments
    --file    <file name>
          Updates the CPU ERoT with the given FW image file.
Optional Arguments
    --individually
          Updates each CPU ERoT with corresponding configuration file
          individually.
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    SFT-DCMS-SINGLE
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c UpdateCpuERot --file ERoT.bin
  In-Band
    # ./saa -I Redfish_HI -u ADMIN -p ADMIN -c UpdateCpuERot --file ERoT.bin
  Remote In-Band
    # ./saa -I Remote_RHI --oi 192.168.34.57 --ou root --op 111111 -u ADMIN -p
        ADMIN -c UpdateCpuERot --file ERoT.bin --remote_saa /root/saa
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c UpdateCpuERot --file
        ERoT.bin
  Remote In-Band
    # ./saa -I Remote_RHI -l IP_ADDR_RANGE.txt -c UpdateCpuERot --file ERoT.bin
Notice
    1. System needs to do AC cycle after CPU ERoT update.
