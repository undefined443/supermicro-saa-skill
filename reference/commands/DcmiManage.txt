Description
    Manages the system by Data Center Manageability Interface (DCMI).
Required Arguments
    --type    <type>
          Manages system with type:
          STD_DCMI
          NM20
Optional Arguments
    None
Usage Modes
    [OOB] [In-Band] [Multiple Systems OOB] [Remote In-Band] [Multiple Systems
    Remote In-Band] 
Node Product Key Required
    No
Examples
Single System
  OOB
    # ./saa -i 192.168.34.56 -u ADMIN -p ADMIN -c DcmiManage --type STD_DCMI -h
  In-Band
    # ./saa -c DcmiManage --type STD_DCMI -h
  Remote In-Band
    # ./saa -I Remote_INB --oi 192.168.34.57 --ou root --op 111111 -c DcmiManage
        --type STD_DCMI -h
Multiple Systems
  OOB
    # ./saa -l IP_ADDR_RANGE.txt -u ADMIN -p ADMIN -c DcmiManage --type STD_DCMI
        -h
  Remote In-Band
    # ./saa -I Remote_INB -l IP_ADDR_RANGE.txt -c DcmiManage --type STD_DCMI -h
